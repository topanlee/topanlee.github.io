Github page+hexo

一月 2, 2017

1、先下载Node、git
2、创建本地文件夹E/Hexo
3、在Hexo文件夹上点击右键 Git bash here,然后输入命令安装hexo了
$ npm install hexo-cli -g #安装Hexo

$ hexo init #在H:\Hexo 文件夹生成blog模板

$ hexo g #生成静态blog == hexo generate

$ hexo s #在本地运行服务器 == hexo server

这一步的前提是有一个github账号
生成新的SSH Key：

$ ssh-keygen -t rsa -C “youremail@youremail.com”（这里是你的邮箱）
回车输入密码再回车确定密码
打开 /Users/your_user_directory/.ssh/id_rsa.pub

命令 hexo g 没有办法成功deploy，
文档说hexo3.0后需要在 _config.yml 中设置 deploy中的type为git（过去是github），
然后，记得一定要在 blog 目录里 执行 npm install hexo-deployer-git –save

将里面的密钥复制
打开Github网站，登陆，进入Account Settings—>SSH Public keys —> add another public keys,点击add key，将复制的密钥粘贴进去

可以通过输入 $ ssh -T git@github.com看看设置是否成功
